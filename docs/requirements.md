# 需求清单

> 所有用户故事按优先级（P0 必做 / P1 重要 / P2 可选）和模块组织。
> 完成的故事在标题前打 `[x]`，进行中打 `[~]`，未开始打 `[ ]`。

## 项目模块

### [ ] US-001: 创建项目（P0）
- **角色**：项目经理
- **故事**：作为项目经理，我希望创建新项目并设置基本信息，以便启动一个新的工作单元
- **关联模块**：pm_project
- **AC**：
  - [ ] 可填写项目名称、描述、开始/截止日期、预算
  - [ ] 自动生成项目编号 PM-YYYY-NNN（年内递增）
  - [ ] 创建后默认状态为「规划中」
  - [ ] 创建人自动成为项目经理（owner）

### [ ] US-002: 项目状态流转（P0）
- **角色**：项目经理
- **故事**：作为项目经理，我希望切换项目状态以反映当前阶段
- **关联模块**：pm_project
- **AC**：
  - [ ] 状态可选：规划中/进行中/已完成/已关闭
  - [ ] 状态变为「已关闭」时，关联未完成任务自动标记为已取消
  - [ ] 已关闭项目不可再编辑（除管理员）

### [ ] US-003: 项目预算与成本（P1）
- **角色**：项目经理
- **故事**：作为项目经理，我希望看到项目预算与实际成本对比
- **关联模块**：pm_project, pm_expense
- **AC**：
  - [ ] `actual_cost` 自动汇总该项目所有 pm_expense 的金额
  - [ ] 项目详情页显示「预算 vs 实际」对比

## 里程碑模块

### [ ] US-010: 里程碑规划（P0）
- **角色**：项目经理
- **故事**：作为项目经理，希望为项目设置里程碑，以便分阶段管理
- **AC**：
  - [ ] 里程碑必须归属一个项目（master_detail）
  - [ ] 可设置里程碑截止日期
  - [ ] 状态：未开始/进行中/已完成
  - [ ] 关联任务全部完成时，里程碑自动变为「已完成」

## 任务模块

### [ ] US-020: 任务管理（P0）
- **角色**：项目成员
- **故事**：作为项目成员，希望创建并跟踪任务
- **AC**：
  - [ ] 任务关联项目和里程碑
